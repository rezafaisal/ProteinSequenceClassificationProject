# ProtSegR

ProtSegR is R Package for Generating Segments and Numerical Representation Schemes for Protein Sequences. This package has 3 main functions:
1. RemoveUnrecognizeAminoAcid.
2. GenerateSegments.
3. ConvertToFeatureRepresentation
